## Использование в Tailwind

Токены `--ds-*` проброшены в `theme.extend` (Stage Prepare, D29) — доступны как утилиты:

| группа | примеры утилит |
| --- | --- |
| Цвета | `bg-bg-base`, `text-text-secondary`, `bg-tech-green`, `border-lines`, `bg-super-yellow` |
| Шрифты | `font-sans` (Grotesk), `font-display` (Instrument Serif), `font-pixel` |
| Кегли | `text-caps`, `text-m`, `text-h3`, `text-description`, `text-h1` |
| Отступы | `p-ds-m`, `gap-ds-s`, `px-ds-l`, `-mt-ds-neg-xs` |
| Радиусы | `rounded-s`, `rounded-m`, `rounded-l`, `rounded-over` (pill) |
| Разрядка | `tracking-caps` · ширина: `max-w-ds-base` |

Источник правды — `src/styles/theme.css` (зеркало remote-библиотеки Figma). Покрытие токенов —
`audit/tokens.coverage.md` (54/54, без битых ссылок).
