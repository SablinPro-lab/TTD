# Canban

**Назначение.** Пайплайн-борд: серифный заголовок + белая карточка с колонками-стадиями,
в каждой — счётчик и список людей (аватар · имя · роль).
Источник правды: Figma → ds-organisms → `canban` (node `1:2048`). **Статус:** stable.

| prop      | значения                                        | по умолч.    |
| --------- | ----------------------------------------------- | ------------ |
| `title`   | `string`                                        | `'Pipeline'` |
| `columns` | `{ title; count?; items: { name; role }[] }[]`  | —            |

Колонки — натуральной figma-ширины (312px); если борд шире области, прокручивается
горизонтально ВНУТРИ карточки (контент не сжимается).
