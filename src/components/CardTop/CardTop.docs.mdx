# CardTop

**Назначение.** Шапка-карточка профиля: акцентный блок с именем/ролью + пилюли-действия + ряды тегов.
Источник правды: Figma → ds-organisms → `card top` (node `1:3744`). **Статус:** stable.

**Фон.** Как в Figma (node `1:3745`): фото (`src/assets/cardtop-bg.jpg`) + слой `mix-blend: color` (чёрный) +
слой `mix-blend: hard-light` (жёлтый `accent`). Карточка 480px, секции распределены `space-between`.
Теги: `tags` (слева) / `tagsRight` (справа), каждый — gold-пилюля с иконкой; кнопка `add` в конце группы.

**Варианты.** `variant='default'` — жёлтая карточка-профиль; `variant='glass'` (Figma node `1:3768`) —
стеклянный hero страницы: широкое фото (`cardtop-glass.jpg`), затухающее в `bg-base`, чёрные пилюли,
слоты `segmented` (сегментный контрол снизу) и `footer`.

| prop          | значения             | по умолч.           |
| ------------- | -------------------- | ------------------- |
| `name`        | `string`             | —                   |
| `role`        | `string`             | —                   |
| `cornerLeft`  | `string`             | —                   |
| `cornerRight` | `string`             | —                   |
| `actions`     | `string[]`           | `[]`                |
| `onAction`    | `(label) => void`    | —                   |
| `accent`      | `string`             | `--ds-color-accent` |

## Живой вариант (нажми пилюлю-действие)

export const Demo = () => {
  const [last, setLast] = useState('')
  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: 12 }}>
      <CardTop
        name="Sarah Mitchell"
        role="Senior software engineer"
        cornerLeft="Teams"
        cornerRight="Access"
        actions={['Promote', 'Negotiate', 'Suspend', 'Fire']}
        onAction={setLast}
      />
      <span style={{ fontSize: 13, opacity: 0.6 }}>последнее действие: {last || '—'}</span>
    </div>
  )
}
