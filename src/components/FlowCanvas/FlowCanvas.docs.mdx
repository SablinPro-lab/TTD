# FlowCanvas

**Назначение.** Канва автоматизаций: ноды (`FlowNode`) можно **перетаскивать** мышью/тачем, а порты —
**соединять «нитками»** (drag от чёрного выходного порта к серому входному другой ноды). Нитки —
SVG-кривые Безье, пересчитываются при перетаскивании. Источник правды: Figma → canvas `1:4253`,
node `1:4254`. **Статус:** stable.

| prop      | значения                                              | по умолч. |
| --------- | ----------------------------------------------------- | --------- |
| `nodes`   | `{ id; title; subtitle?; color?; x; y }[]`            | —         |
| `edges`   | `{ from; to }[]` (выход→вход)                          | `[]`      |
| `onSelect`| `(id) => void` (клик/перетаскивание ноды)             | —         |

**Взаимодействие.** Перетаскивание ноды — pointerdown по телу. Соединение — pointerdown по правому
(чёрному) порту → тянем к левому (серому) порту другой ноды → отпускаем. Оптимизация: слушатели на
`window` + троттлинг `requestAnimationFrame`, позиционирование через CSS `transform`.
